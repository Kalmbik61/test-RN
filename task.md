Тестовое задание 1 - Mecenate

Платформа: iOS + Android (React Native + Expo)

Задача

Разработать экран ленты публикаций для платформы Mecenate — сервиса поддержки
авторов (аналог Patreon/Boosty).
Пользователь открывает приложение и видит список постов от авторов, на которых
подписан.

Что нужно реализовать

Экран Feed (лента)
● Список постов: аватар автора, имя, превью текста, обложка поста, счётчик лайков и
комментариев
● Пагинация — курсорная (подгрузка при скролле вниз)
● Pull-to-refresh
● Закрытый пост (tier: "paid") — показывать заглушку вместо текста
● Сообщение об ошибке если API недоступен: «Не удалось загрузить публикации»
с кнопкой повтора

Тестовое задание 2 — Mecenate

Платформа: iOS + Android (React Native + Expo)

Задача

Разработать экран детальной публикации для платформы Mecenate с real-time
обновлениями и интерактивными элементами.
Пользователь нажимает на пост — переходит на детальный экран, может лайкнуть,
почитать комментарии, и видит обновления в реальном времени.

Что нужно реализовать

Экран Feed (лента)
● Список постов: аватар, имя автора, превью, обложка
● Таб-фильтр над списком: Все / Бесплатные / Платные
● Tap на пост → переход на детальный экран
Экран Post Detail
● Полный текст публикации, обложка, имя автора
● Кнопка лайка с анимацией (Reanimated 2): плавное увеличение счётчика + haptic
feedback
● Список комментариев с Lazy Load
● Поле ввода + кнопка отправки нового комментария
● Real-time: новые лайки и комментарии появляются без перезагрузки (WebSocket)
